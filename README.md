# Clock-Out-SOEM

A simple Python tool to calculate clock-out times and OT.

## How to Run This Online (Free)
1. Click the green **Code** button at the top right.
2. Select the **Codespaces** tab.
3. Click **Create codespace on main**.
4. Wait for the terminal to open at the bottom.
5. Type the following and press Enter:
   ```bash
   python Clock_out.py
